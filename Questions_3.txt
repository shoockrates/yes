„Nupiešti (rankomis)“ čia suprantame tokiu būdu: jei norime piešti tokį

    lapcia
   ┌───┴───┐
 apcia  tapcia
       ┌───┴───┐
     muska   tuzka

medį, tai „piešimo“ procesą užduotėlės sprendime užrašome:

lapcia -> apcia;  // left
lapcia -> tapcia; // right
tapcia -> muska;  // left
tapcia -> tuzka;  // right
